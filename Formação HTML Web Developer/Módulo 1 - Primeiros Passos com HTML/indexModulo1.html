Grandes Nomes da Tecnologia

(Voltar)

O curso da plataforma Dio começa explicando a história dos computadores e de como surgiu a Internet.

Para ser possível estarmos conectados a tudo que acontece ao redor do mundo hoje, existiram pessoas que tiveram o papel essencial para chegarmos até aqui. São elas:

Grace Hopper, criadora da linguagem Flow-Matic, foi uma das primeiras programadoras do Mark I, e popularizou o termo "bug" na computação.

Robert E. Kahn, criador do TCP/IP, ele realizou a primeira apresentação pública da ARPANet.

Tim Berners-Lee, Inventor do WWW, responsável por popularizar o HTTP e HTML.

Marc Andreessen, Criou o navegador Netscape, feito a partir do Mosaic. Ele implementou recursos gráficos ao navegador, deixando o mesmo mais amigável para o usuário.

E por último, nomes mais recentes, como Bill Gates e Steve Jobs, que fundaram gigantes da tecnologia (Microsoft e Apple, respectivamente), agiram para popularizar os computadores e smartphones para todas as pessoas.

Client x Server

(Voltar)

Basicamente, o conceito de Client pode ser explicado por ser o lado do usuário que acessa algo na internet.

A partir de um computador ou smartphone, esse usuário acessa a internet por um navegador, e vai acessar sites (utilizando o protocolo HTTP) que estão hospedados em servidores.

A partir dos navegadores, o usuário vai ter acesso a informações na tela, que consistem em dados que podem ser HTML, CSS, Javascript, Imagens, etc...

Já o conceito de Server, representa o outro lado, oque guarda os dados que o usuário está acessando. Os servidores também podem estar conectados a internet, e quando um usuário tenta acessar alguma coisa, esse servidor vai enviar os dados armazenados, que foram requisitados.

Um mesmo servidor pode enviar dados para vários usuários diferentes, de vários lugares diferentes. Existe também a possibilidade de existir dados de um servidor local, onde não é necessário estar conectado a internet para usuário que fazem parte dessa mesma rede (LAN), conseguirem acessar os dados armazenados.

Linguagens de Programação

(Voltar)

Existem diversas linguagens de programação hoje em dia, mas podemos listar aqui as 5 primeiras, segdundo o RedMonk:

  1. JavaScript
  2. Python
  3. Java
  4. PHP
  5. C#

Podemos classificar as linguagens com dois tipos diferentes, sendo eles Server-Side (ou Back-End), ou também Client-Side (ou Front-End).

Cada linguagem tem as suas características e finalidades diferentes. As linguagenS Client-Side, são executadas do lado do usuário, e um exemplo de uma, é o Javascript, rodando pelo navegador.

Já as linguagens Server-Side, trabalham com a estrutura "por trás dos panos", podendo ser Java, C#, e também, o próprio Javascript, atuando via NodeJS.

Introdução ao HTML

(Voltar)

Parte do curso da DIO, consiste em explicar um pouco de como funciona o HTML. Conhecemos ferramentas que vamos trabalhar, como o VisualStudio Code e o console do prórpio navegador.

Aprendemos um pouco sobre estrutura básica do HTML, como montar uma página simples, e como funcionam as tags e seus atributos.

Aprendemos também um pouco de como formatar textos, criar listas ordenadas ou não ordenadas, e até como adicionar links.

O mais legal, é que ao final das aulas, estamos capacitados a criar um site simples, porém funcional, exatamente como esse aqui, que foi feito utilizando apenas o conhecimento passado nesse primeiro módulo do curso!
